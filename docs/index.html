01 Просмотр сейфа

Идентификатор вашего сейфа в контракте

Сборка EIP-712-сообщений для всех операций сейфа. Подпишите ключом сейфа в любом кошельке, потом получатель отправляет транзакцию.

A Withdraw

Регулярный вывод из сейфа. Подпись mainAuthKey.

B Transfer

Перевод между сейфами с таймлоком. mainAuthKey.

C Early Close (5%)

Досрочное закрытие со штрафом 5%. recoveryAuthKey.

D Recover to Safe (10%)

Эвакуация на снапшот emergencyAddress. recoveryAuthKey.

E Emergency → Any (15%)

Вывод на любой адрес со штрафом 15%. recoveryAuthKey.

F Rotate Auth Keys

Смена обоих ключей авторизации. recoveryAuthKey.

G Set Timelock

Изменение таймлока (зависит от уровня). mainAuthKey.

H Voluntary Lock

Добровольная блокировка вывода до даты. mainAuthKey.

Panic Withdraw (без подписи)

Мгновенная эвакуация 80% на live globalEmergency. Штраф 20%. Подпись НЕ нужна, но вызвать может только владелец сейфа.
